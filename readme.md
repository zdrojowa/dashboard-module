## Installation

Via Composer

``` bash
$ composer require zdrojowa/dashboardmodule
```

NPM required:
``` bash
"axios": "^0.19",
"bootstrap": "^4.5.3",
"bootstrap-vue": "2.16.0",
"vue": "^2.6.10",
"vue-router": "^3.4.9",
```

## Usage
1. Add mix.module('DashboardModule', 'vendor/zdrojowa/dashboardmodule'); in webpack.mix.js
2. npm run dev
